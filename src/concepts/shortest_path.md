# 최단 경로 (Dijkstra / Floyd / Bellman-Ford / 0-1 BFS)

## 한 줄 요약
가중치 그래프에서 최소 비용 경로. 상황에 맞는 알고리즘 선택이 핵심.

## 어떤 걸 언제
- 한 점에서 전체, 음수 없음 → 다익스트라 (우선순위 큐) O(E log V)
- 모든 쌍 → 플로이드 워셜 O(V^3) (V 작을 때)
- 음수 간선 가능 / 음수 사이클 탐지 → 벨만-포드 O(VE)
- 가중치가 0 또는 1뿐 → 0-1 BFS(deque) O(V+E)

## 왜 이거 + 대안 + 차이
- 다익스트라는 음수 간선이 있으면 틀린다 → 그땐 벨만-포드.
- 0-1 BFS는 다익스트라보다 빠름(가중치 0/1 한정): 0이면 push_front, 1이면 push_back.

## 핵심
- 다익스트라: dist 초기 무한, 더 짧을 때만 갱신/push, pop 시 stale(거리 큰) 건너뛰기.
- INF 더할 때 오버플로 주의(1e18 등).

## 흔한 실수
- 음수 간선에 다익스트라 적용
- 우선순위 큐에서 이미 처리한 정점 재방문 미차단
