# 스택 (Stack)

## 한 줄 요약
나중에 넣은 게 먼저 나오는(LIFO) 자료구조. "가장 최근 것"과 짝/비교할 때 강력.

## 언제 쓰나
- 괄호 짝 맞추기 (VPS, 쇠막대기)
- 직전 원소와의 관계 (오큰수, 탑 — 단조 스택)
- 함수 호출/되돌리기, 후위표기식 계산

## 왜 이거 + 대안 + 차이
- 큐(FIFO)와 반대. "가장 최근"이 중요하면 스택, "가장 오래된"이 중요하면 큐.
- 단조 스택: 오큰수/탑처럼 "다음에 더 큰/작은 원소"를 O(N)에. 매번 비교하면 O(N²).

## C++ 핵심
- `stack<int> st; st.push(x); st.top(); st.pop(); st.empty();`
- 단조 스택은 보통 인덱스를 담는다.

## 흔한 실수
- empty() 체크 없이 top()/pop() → 런타임 에러
- 단조 스택에서 push/pop 조건(< 인지 <= 인지) 헷갈림
